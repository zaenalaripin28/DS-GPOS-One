Komponen
Cara pakai token di Mantine

Import CSS variables lalu gunakan langsung di Mantine theme override atau inline style:

// di theme.ts export const theme = createTheme({ colors: { brand: [ 'var(--color-blue-B50)', 'var(--color-blue-B75)', 'var(--color-blue-B100)', 'var(--color-blue-B200)', 'var(--color-blue-B300)', 'var(--color-blue-B400)', 'var(--color-blue-B500)', ] } })
// di Tailwind config module.exports = { theme: { extend: { colors: { 'brand': 'var(--color-blue-B300)', 'brand-hover': 'var(--color-blue-B400)', 'surface': 'var(--color-neutral-N10)', }}} }
Install CSS tokens

Copy semua token ke file tokens.css lalu import di _app.tsx:

/* tokens.css - paste semua :root variables */ :root { --color-blue-B300: #1E7FD6; --color-blue-B400: #155996; --space-200: 16px; --radius-100: 8px; /* ... lihat halaman Token Reference */ }
