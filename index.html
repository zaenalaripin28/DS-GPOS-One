Cara pakai token di Tailwind

Import CSS variables ke tokens.css, lalu gunakan di Tailwind config dan class HTML:

// tailwind.config.js — extend dengan GPOS tokens module.exports = { theme: { extend: { colors: { 'brand': 'var(--color-blue-B300)', 'brand-hover': 'var(--color-blue-B400)', 'brand-light': 'var(--color-blue-B50)', 'danger': 'var(--color-red-R300)', 'success': 'var(--color-green-G300)', 'warning': 'var(--color-orange-O300)', 'surface': 'var(--color-neutral-N10)', 'n900': 'var(--color-neutral-N900)', 'n500': 'var(--color-neutral-N500)', 'n300': 'var(--color-neutral-N300)', 'n40': 'var(--color-neutral-N40)', }, borderRadius: { 'gpos-sm': 'var(--radius-050)', 'gpos': 'var(--radius-100)', 'gpos-lg': 'var(--radius-150)', }, spacing: { 'gpos-1': 'var(--space-100)', 'gpos-2': 'var(--space-200)', 'gpos-3': 'var(--space-300)', 'gpos-4': 'var(--space-400)', } } } }
// Contoh penggunaan di JSX/HTML <button className="bg-brand hover:bg-brand-hover text-white h-10 px-4 rounded-gpos font-medium transition-colors duration-150"> Primary Button </button> <span className="bg-brand-light text-brand text-xs font-medium px-3 py-1 rounded-full"> Badge Info </span>
Install CSS tokens

Copy semua token ke file tokens.css lalu import di _app.tsx:

/* tokens.css - paste semua :root variables */ :root { --color-blue-B300: #1E7FD6; --color-blue-B400: #155996; --space-200: 16px; --radius-100: 8px; /* ... lihat halaman Token Reference */ }
